# eFfEctiVe_jAVa
이펙티브 자바라는 책은 내용이 굉장히 좋지만, 무조건적으로 수용하면서 책을 읽으면 위험하다는 이야기를 많이 들었습니다. 
또한, 많은 블로그들을 보면 책의 내용을 그대로 복사 붙여넣기 하는 경우가 많았습니다.

저희 스터디는 그렇게 책을 읽고싶지 않아 비판적으로 책을 읽고 발표하고 토론하면서 생각을 정리해 나갈 생각이고, 이를 이 레포지토리에 정리하려고합니다.

주 2회 스터디 진행
